Enhancement One: Software Design & Engineering

For this milestone, I picked the artifact Grand Strand Systems from my CS-320 course. The application uses the Java programming language. The goal of the project is to create an application to schedule appointments, collect user information and provide a list of tasks to be completed. Automated tested is preformed by using JUnit testing to confirm if the given information is corect and does not result in a type or overflow error. Since the assignment was done early in my programming history it may have contained minor issues such as mislabel methods and variables. The enhanchcement hence aims to improve the overall readability of the application by fixing the inconsistencies found in the structure of the code. The enhanced version also features fixes to the test modules as the original artifact would not have ran them without errors.
